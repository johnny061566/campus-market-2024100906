## 2. 学习收获

通过今天的学习，我对 Vue 3 工程化项目有了更清晰的认识。

我理解到 `main.ts` 是项目的入口文件，负责把 `App.vue`、Pinia 和 Router 连接起来。`App.vue` 中的 `<RouterView />` 是页面显示的位置，路由匹配到的组件会显示在那里。`router/index.ts` 负责维护路径和页面组件之间的对应关系，后续开发多个页面时需要不断扩展这个文件。

我也认识到，一个前端项目不只是写页面，还包括依赖管理、类型检查、代码规范、项目文档和过程证据。`docs` 目录虽然不是业务代码，但它能记录开发过程，对实训验收很重要。
